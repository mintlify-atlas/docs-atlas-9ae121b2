The `env` package gives Go applications a clean, consistent API for reading environment variables as typed values. Instead of calling `os.Getenv` and manually parsing strings, you call `env.Int("PORT", 8080)` and get an `int` back â€” with a fallback if the variable is missing or unparseable.

## Installation

```bash
go get -u github.com/andreimerlescu/goenv/env
```

## Import paths

The `env` package lives inside the `goenv` module as a subpackage:

```go
// Subpackage â€” recommended for library use in your application
import "github.com/andreimerlescu/goenv/env"
```

Use this import when you want just the environment-variable library without the CLI. Every function is called as `env.String(...)`, `env.Int(...)`, etc.

<Note>
  The `env` package reads from the process runtime environment. To load a `.env` file into the environment before your application starts, use the `goenv` CLI with your shell's source mechanism.
</Note>

## The Magic() function

`Magic()` reads all `AM_GO_ENV_*` environment variables at startup and uses them to configure the package's behavior â€” things like separators, panic policy, and logging. Call it once from your application's `init()` function to apply any environment-level overrides before other code runs.

```go
func init() {
    env.Magic()
}
```

When `Magic()` runs, it reads variables like `AM_GO_ENV_LIST_SEPARATOR`, `AM_GO_ENV_ALWAYS_ALLOW_PANIC`, and `AM_GO_ENV_ALWAYS_USE_LOGGER` and assigns them to the corresponding package-level variables. This lets operators control package behavior through environment variables alone, without recompiling.

See [Package configuration](/library/configuration) for the full list of variables `Magic()` reads.

## Full working example

The following program demonstrates type functions, collections, duration parsing, and package configuration together:

```go
package main

import (
    "fmt"
    "strings"
    "time"

    "github.com/andreimerlescu/goenv/env"
)

func init() {
    // Apply any AM_GO_ENV_* environment variables to configure the package.
    env.Magic()

    // Require critical variables at startup.
    env.MustExist("DATABASE_URL")
}

func main() {
    // Scalar types â€” returns fallback if the variable is unset or unparseable.
    hostname := env.String("HOSTNAME", "localhost")
    port     := env.Int("PORT", 3306)
    username := env.String("USERNAME", "user")
    password := env.String("PASSWORD", "")
    pi64     := env.Float64("PI", 3.14159265358979)
    pi32     := env.Float32("PI", 3.14159)
    debug    := env.Bool("DEBUG", false)

    // Duration â€” accepts "300ms", "1.5h", or nanosecond integers.
    timeout := env.Duration("TIMEOUT", 5*time.Second)

    // UnitDuration â€” TIMEOUT=10 with unit=time.Second means 10 seconds.
    readTimeout := env.UnitDuration("READ_TIMEOUT", 3, time.Second)

    // Collections â€” comma-separated by default.
    tags       := env.List("TAGS", env.ZeroList)
    properties := env.Map("CONFIG", env.ZeroMap)

    fmt.Println(hostname, port, username, strings.Repeat("*", len(password)))
    fmt.Println("pi64:", pi64, "pi32:", pi32, "debug:", debug)
    fmt.Println("timeout:", timeout, "readTimeout:", readTimeout)
    fmt.Println("tags:", strings.Join(tags, ", "))
    fmt.Println("properties:", properties)
}
```
